最近 glance-chamburr 显示 markdown 很不靠谱，老掉内容，我也懒得修它了，卸载算了。

```
brew remove glance-chamburr  

qlmanage -r  

qlmanage -r cache  
```

然后按住 Option 键右键点击 Dock 上的 Finder 图标，选"重新开启"重开 Finder 即可。
